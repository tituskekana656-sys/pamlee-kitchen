# Pam_Lee's Kitchen - Static HTML/CSS/JS Version

This is a vanilla HTML/CSS/JavaScript conversion of the Pam_Lee's Kitchen bakery website.

## Features

- ✅ Fully responsive design
- ✅ Mobile-friendly navigation
- ✅ Shopping cart with localStorage
- ✅ Product filtering by category
- ✅ Beautiful animations and transitions
- ✅ SEO-optimized HTML structure

## Pages Included

- `index.html` - Home page with hero section and featured products
- `menu.html` - Full product menu with category filters
- `gallery.html` - Photo gallery
- `contact.html` - Contact information and form
- `login.html` - Demo login page
- `admin.html` - Demo admin dashboard

## JavaScript Files

- `script.js` - Main UI interactions (mobile menu, animations)
- `cart.js` - Shopping cart functionality with localStorage
- `menu.js` - Product data and filtering

## Limitations (Static Version)

This is a static demo with no backend:
- ❌ No real database (products are hardcoded)
- ❌ No user authentication
- ❌ No real checkout/payment processing
- ❌ No admin functionality
- ❌ No order management

## How to Use

1. Open `index.html` in a web browser
2. Browse products and add items to cart
3. Cart data is saved in browser localStorage
4. All navigation works via standard HTML links

## To Deploy

Upload all files to any web hosting service:
- Netlify
- Vercel
- GitHub Pages
- Any static hosting provider

## Design System

The site uses a gold and cream luxury bakery theme with:
- Primary: Dark navy (#0f172a)
- Secondary: Gold (#d4a574)
- Fonts: Playfair Display (headings), Poppins (body)

All styles are in `styles.css` - no external CSS frameworks required.

Works on all modern browsers:

- Firefox
